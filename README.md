# タイムテーブル計算機くん

イベント用タイムテーブルを簡単に作成できるWebアプリです。

DJイベント、ライブイベント、クラブイベント、学園祭などで使用できます。

## 主な機能

* DJ順のドラッグ＆ドロップ並び替え
* レギュラーDJ人数設定
* ゲスト出演者対応
* ライブ出演者対応
* 出演時間の自動計算
* タイムテーブル画像保存
* スマホ対応


## 使い方

### 1. イベント情報を入力

* イベント名
* 日付
* 開始時間
* 終了時間

### 2. DJを入力

DJ名を入力し、ドラッグして順番を並び替えます。

### 3. ゲスト・ライブを設定

* 名前
* 人数
* 持ち時間
* 開始時間

を設定できます。

### 4. タイムテーブル生成

「生成」を押すとタイムテーブルを作成します。

### 5. 画像保存

「画像保存」を押すと画像で保存できます。


## ホーム画面に追加

本アプリはスマートフォンのホーム画面に追加して、アプリのように使用できます。

### iPhone（Safari）

1. 公開URLをSafariで開く
2. 画面下部の共有ボタンをタップ
3. 「ホーム画面に追加」を選択
4. 「追加」をタップ

### Android（Chrome）

1. 公開URLをChromeで開く
2. 右上のメニュー（︙）をタップ
3. 「ホーム画面に追加」を選択
4. 「インストール」または「追加」をタップ

ホーム画面から直接起動できるため、イベント現場でも素早く利用できます。
